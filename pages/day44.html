Design: Detectors as Plugins

Each detector implements a simple contract: it inspects a frame and optionally emits alerts. A central engine feeds every frame to every registered detector. Adding a new attack signature is just writing a new class — the core never changes.

Separation of concerns wins.
Capture, detection, and reporting are independent layers. You can swap live sniffing for a PCAP file, add or remove detectors, and change output format (console, JSON, syslog) without rewriting anything else. This is how production WIDS tools are structured.

The Framework

python wids_engine.py 
import time
from dataclasses import dataclass, field
from collections import defaultdict
from scapy.all import Dot11, Dot11Beacon, Dot11Deauth, Dot11Elt

@dataclass
class Alert:
    kind: str
    detail: str
    ts: float = field(default_factory=time.time)

class Detector:
    """Base class: override inspect() to return a list of Alerts."""
    def inspect(self, pkt):
        return []

class DeauthFlood(Detector):
    def __init__(self, window=10, threshold=20):
        self.window, self.threshold = window, threshold
        self.hits = defaultdict(list)
    def inspect(self, pkt):
        if not pkt.haslayer(Dot11Deauth):
            return []
        now, src = time.time(), pkt.addr2
        self.hits[src] = [t for t in self.hits[src] if now - t < self.window]
        self.hits[src].append(now)
        if len(self.hits[src]) > self.threshold:
            return [Alert("DEAUTH_FLOOD", f"{src} "
                          f"{len(self.hits[src])}/{self.window}s")]
        return []

class RogueAP(Detector):
    def __init__(self, allowlist):
        self.allow = allowlist              # {ssid: {bssid, ...}}
    def inspect(self, pkt):
        if not pkt.haslayer(Dot11Beacon):
            return []
        ssid = pkt[Dot11Elt].info.decode(errors="ignore").lower()
        if ssid in self.allow and pkt.addr3 not in self.allow[ssid]:
            return [Alert("ROGUE_AP", f"'{ssid}' from {pkt.addr3}")]
        return []

class Engine:
    def __init__(self, detectors):
        self.detectors = detectors
    def feed(self, pkt):
        for d in self.detectors:
            for alert in d.inspect(pkt):
                print(f"🚨 {time.strftime('%H:%M:%S', time.localtime(alert.ts))}"
                      f"  [{alert.kind}] {alert.detail}")

# Wiring
if __name__ == "__main__":
    from scapy.all import sniff
    engine = Engine([
        DeauthFlood(),
        RogueAP({"corp-wifi": {"aa:bb:cc:00:11:22"}}),
    ])
    sniff(iface="wlan0mon", prn=engine.feed, store=0)

🧠 Knowledge Check
What is the benefit of pluggable detectors?

You add signatures without touching the engine, and keep capture/detect/report separate.

What contract does each detector implement?

inspect(pkt) returning a list of alerts.

Why emit JSON output?

So alerts feed a SIEM without changing the detectors.
