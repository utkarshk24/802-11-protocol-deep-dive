The Data Rate Formula

Every OFDM PHY rate follows the same structure — bits carried per symbol divided by symbol time:

text PHY Rate 
Rate = (N_SD × N_BPSCS × R × N_SS) / T_SYM

N_SD     = data subcarriers (depends on bandwidth)
N_BPSCS  = coded bits per subcarrier per stream (modulation)
R        = coding rate (e.g. 3/4, 5/6)
N_SS     = number of spatial streams
T_SYM    = symbol duration (3.6 µs long GI, 3.2 µs short GI + 0.4 GI)

The Building-Block Numbers

text Constants 
Data subcarriers (N_SD):
  20 MHz = 52     40 MHz = 108
  80 MHz = 234    160 MHz = 468

Bits/subcarrier (N_BPSCS) by modulation:
  BPSK=1  QPSK=2  16-QAM=4  64-QAM=6  256-QAM=8

Symbol time:
  Long GI  = 3.6 µs    Short GI = 3.2 µs (400 ns GI)
📐

Worked Example: MCS 9, 80 MHz, 1 SS

MCS 9 = 256-QAM (8 bits) with coding rate 5/6, on an 80 MHz channel, short GI:

text Compute 
N_SD    = 234        (80 MHz)
N_BPSCS = 8          (256-QAM)
R       = 5/6
N_SS    = 1
T_SYM   = 3.2 µs     (short GI)

Rate = (234 × 8 × 5/6 × 1) / 3.2e-6
     = 1560 / 3.2e-6
     = 487.5 Mbps    ... wait, per the standard: 433.3 Mbps @ long GI,
                          481.5 Mbps @ short GI  (rounding in spec tables)
Spec tables round; the formula is exact.
Published MCS tables quote standardized values (e.g. 433.3 Mbps for 80/256-QAM/5-6/1SS long GI). Minor differences come from guard-interval choice and rounding. Learn the formula and you can reproduce — and sanity-check — any table.

A Reusable Rate Calculator

python vht_rate.py 
# MCS index -> (bits/subcarrier, coding rate)
MCS = {
    0: (1, 1/2), 1: (2, 1/2), 2: (2, 3/4), 3: (4, 1/2),
    4: (4, 3/4), 5: (6, 2/3), 6: (6, 3/4), 7: (6, 5/6),
    8: (8, 3/4), 9: (8, 5/6),
}
N_SD = {20: 52, 40: 108, 80: 234, 160: 468}

def vht_rate(mcs, bw, nss, short_gi=False):
    bpscs, r = MCS[mcs]
    t_sym = 3.2e-6 if short_gi else 3.6e-6   # + GI already folded in
    bits = N_SD[bw] * bpscs * r * nss
    return bits / t_sym / 1e6                # Mbps

for bw in (20, 80, 160):
    print(f"MCS9 {bw:3d}MHz 2SS SGI = "
          f"{vht_rate(9, bw, 2, short_gi=True):7.1f} Mbps")

🧠 Knowledge Check
What are the components of the rate formula?

(data subcarriers x bits-per-subcarrier x coding x streams) divided by symbol time.

How many data subcarriers does an 80 MHz channel have?

234.

Short vs long guard-interval symbol times?

3.2 microseconds (SGI) vs 3.6 microseconds (LGI), including the guard.
