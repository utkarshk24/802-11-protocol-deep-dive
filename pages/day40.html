The Graveyard: WEP & TKIP

  • WEP: RC4 with a 24-bit IV. IV reuse and the FMS attack let anyone recover the key in minutes. Utterly broken; never use.
  • TKIP: A 2003 stopgap that wrapped RC4 with per-packet key mixing and a MIC (Michael) to run on WEP-era hardware. It patched WEP's worst flaws but is itself now insecure (Beck-Tews and later attacks). WPA3 forbids TKIP entirely.
TKIP's presence downgrades a whole network.
If an AP still advertises TKIP for backward compatibility, it can force the group cipher down to TKIP even for CCMP-capable clients. Modern security guidance: disable TKIP everywhere and run CCMP-only (or GCMP).
🔐

The Modern Standards: CCMP & GCMP

Both are AES-based AEAD modes that encrypt and authenticate every frame, with a per-packet nonce (Packet Number) that prevents replay:

text Cipher Comparison 
Cipher     Base   Auth        Key    Used by
WEP        RC4    CRC (!)     40/104 dead
TKIP       RC4    Michael     128    deprecated (WPA-TKIP)
CCMP-128   AES    CBC-MAC     128    WPA2, WPA3-Personal/Ent base
GCMP-128   AES    GMAC        128    (optional)
GCMP-256   AES    GMAC        256    WPA3-Enterprise 192-bit

The Packet Number in each CCMP/GCMP header is a monotonically increasing counter; a receiver rejects any frame whose PN it has already seen — the built-in replay defense.

🔓

OWE — Encryption Without a Password

Traditional "open" networks (coffee shops, airports) send everything in the clear — anyone can sniff your traffic. Opportunistic Wireless Encryption (OWE), marketed as Enhanced Open, fixes this without adding a password:

  • Client and AP perform an unauthenticated Diffie-Hellman exchange during association (carried in an OWE Diffie-Hellman Parameter element).
  • They derive a unique pairwise key, so each client's traffic is encrypted with a distinct key.
  • The user experience is unchanged — still "no password" — but passive eavesdropping is defeated.
OWE stops passive sniffing, not active MITM.
Because there's no authentication, an active attacker running an evil twin can still interpose. OWE's win is that casual eavesdropping on an open network no longer works — a big improvement over cleartext, but not a substitute for authenticated WPA3.
