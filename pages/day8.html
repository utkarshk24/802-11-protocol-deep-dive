Capstone Project: PCAP Analyzer

Requirements

Build a Python tool that reads a PCAP file and generates a report with:

  • Frame counts: Total frames, management, control, data
  • Channel analysis: Channels used, frequencies
  • RSSI histogram: Distribution of signal strengths
  • Data rates: Which MCS rates were used
  • BSSIDs found: List of APs and beacon count per AP
  • Device tracking: Devices and their connection attempts
  • Anomalies: Deauths, disassocs, failed auths, corrupted frames
