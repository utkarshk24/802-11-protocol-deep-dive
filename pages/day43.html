Attacks That Survive WPA3

  • Evil twin / rogue AP: beacons aren't authenticated, so a clone SSID is always possible. Detect via BSSID allowlists and RF fingerprinting.
  • Downgrade / transition-mode abuse: if a network runs WPA2/WPA3 mixed mode, an attacker forces a client to the weaker WPA2 path. Detect by watching for unexpected WPA2 associations to a WPA3-capable SSID.
  • SAE flooding / anti-clogging bypass: a burst of SAE commit frames aimed at exhausting the AP.
  • Beacon/CSA spoofing: forged Channel Switch Announcements to herd clients onto an attacker's channel (unprotected in beacons).
  • Karma / MANA: an AP that answers every probe with a matching SSID to lure clients.
📉

Downgrade Detection

Transition mode (WPA3+WPA2 on one SSID) is the pragmatic reality during migration, but it's a downgrade surface. A monitor should flag when a client that advertised SAE capability ends up completing a plain PSK 4-way handshake to a known WPA3 SSID — a sign it was pushed down.

Transition mode is a trade-off, not a bug.
It exists so old devices still connect. But it means WPA3's protections are only as strong as the weakest allowed path. High-security networks should run WPA3-only; detection tooling should alert on downgrades in mixed environments.

Detection Heuristics in Code

python protocol_ids.py 
from scapy.all import sniff, Dot11, Dot11Beacon, Dot11Elt, Dot11Auth
from collections import defaultdict
import time

# Known-good: SSID -> allowed BSSIDs, and expected security
BASELINE = {
    "corp-wifi": {"bssids": {"aa:bb:cc:00:11:22"}, "wpa3": True},
}
sae_commits = defaultdict(list)

def alert(kind, detail):
    print(f"🚨 [{kind}] {detail}")

def rsn_has_sae(pkt):
    # AKM suite type 8 (SAE) appears in the RSN element (ID 48)
    elt = pkt.getlayer(Dot11Elt)
    while elt:
        if elt.ID == 48 and b"\x00\x0f\xac\x08" in bytes(elt.info):
            return True
        elt = elt.payload.getlayer(Dot11Elt)
    return False

def handle(pkt):
    if not pkt.haslayer(Dot11):
        return

    if pkt.haslayer(Dot11Beacon):
        ssid = pkt[Dot11Elt].info.decode(errors="ignore").lower()
        bssid = pkt.addr3
        base = BASELINE.get(ssid)
        if base and bssid not in base["bssids"]:
            alert("ROGUE-AP", f"'{ssid}' beaconed by {bssid}")
        if base and base["wpa3"] and not rsn_has_sae(pkt):
            alert("DOWNGRADE", f"'{ssid}' beacon missing SAE AKM ({bssid})")

    # SAE commit flood (Auth alg=3, seq=1)
    if pkt.haslayer(Dot11Auth) and pkt[Dot11Auth].algo == 3 \
            and pkt[Dot11Auth].seqnum == 1:
        now = time.time()
        src = pkt.addr2
        sae_commits[src] = [t for t in sae_commits[src] if now - t < 5]
        sae_commits[src].append(now)
        if len(sae_commits[src]) > 30:
            alert("SAE-FLOOD", f"{src}: {len(sae_commits[src])} commits/5s")

sniff(iface="wlan0mon", prn=handle, store=0)
