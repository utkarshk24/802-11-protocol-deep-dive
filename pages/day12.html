Association Request (Subtype 0)

The client's Association Request advertises what it can do and which network it wants. Fixed fields plus IEs:

  • Capability Info (2 bytes): ESS, Privacy, Short Preamble, Spectrum Mgmt, QoS support.
  • Listen Interval (2 bytes): How often (in beacon intervals) the station wakes to check for buffered traffic — the AP uses this to size power-save buffers.
  • SSID IE: The exact network the client wants to join.
  • Supported/Extended Rates: Data rates the client supports.
  • RSN IE: Chosen pairwise cipher, AKM suite (PSK, SAE, 802.1X), and group cipher — this must be a subset of the AP's advertised RSN.
  • HT/VHT/HE Capabilities: The client's PHY abilities so the AP knows the fastest common rate.
🆔

Association Response (Subtype 1)

The AP replies with a status code and, on success, an Association ID:

  • Capability Info: The AP's capabilities (echoing what the BSS supports).
  • Status Code: 0 = success. Common rejections: 17 (AP can't handle more stations), 18 (rates mismatch), 40–45 (invalid IE / cipher / AKM).
  • Association ID (AID): A 14-bit identifier (1–2007) the AP assigns. The AID indexes the station's bit in the TIM for power-save buffering.
The AID is how power save works.
When the AP buffers frames for a sleeping station, it sets that station's AID bit in the beacon's TIM element. The station wakes, sees its bit, and sends a PS-Poll to retrieve the buffered frames. No AID, no targeted power-save delivery.
python assoc_parser.py 
from scapy.all import rdpcap, Dot11, Dot11AssoReq, Dot11AssoResp, Dot11Elt

for pkt in rdpcap("capture.pcap"):
    if pkt.haslayer(Dot11AssoReq):
        ssid = pkt[Dot11Elt].info.decode(errors="ignore")
        print(f"[REQ]  {pkt[Dot11].addr2} wants SSID '{ssid}' "
              f"listen_int={pkt[Dot11AssoReq].listen_interval}")
    elif pkt.haslayer(Dot11AssoResp):
        r = pkt[Dot11AssoResp]
        aid = r.AID & 0x3fff
        result = "OK" if r.status == 0 else f"REJECT({r.status})"
        print(f"[RESP] {pkt[Dot11].addr2} -> {pkt[Dot11].addr1}  "
              f"{result}  AID={aid}")
