Detecting Attacks
via Frame Analysis

Management frames are unauthenticated by default, which makes them the raw material for most Wi-Fi attacks. Today we build a detection engine that spots deauth floods, evil twins, MAC spoofing, and rogue APs — the defensive core of a wireless IDS.

The Attack Catalog

  • Deauth/Disassoc flood: Spoofed teardown frames knock clients offline. Signature: high rate of deauth from a BSSID.
  • Evil twin: A rogue AP clones a legitimate SSID (often the same or spoofed BSSID) to lure clients. Signature: same SSID on unexpected channel, mismatched capabilities, or a new BSSID for a known SSID.
  • MAC spoofing: An attacker impersonates a client or AP MAC. Signature: sudden change in sequence-number progression or signal strength for a known MAC.
  • Rogue AP: An unauthorized AP on the corporate wire. Signature: a BSSID not in the approved allowlist beaconing your SSID.
  • Karma / beacon flood: Responding to every probe / flooding fake SSIDs. Signature: implausible number of distinct SSIDs from one radio.
👯

Evil Twin & Spoofing Heuristics

Two powerful signals: an SSID you trust appearing on a new BSSID, and a known BSSID whose sequence numbers suddenly jump (802.11 sequence numbers increment monotonically per transmitter — a spoofer's chip starts from its own counter).

Sequence numbers are a spoofing fingerprint.
Each real transmitter increments its 12-bit sequence counter by 1 per frame. If you see the same BSSID emit frames whose sequence numbers move backward or leap by thousands, two radios are using that MAC — a strong spoofing indicator.
🐍

A Minimal Detection Engine

python wids.py 
from scapy.all import sniff, Dot11, Dot11Beacon, Dot11Deauth, Dot11Elt
from collections import defaultdict
import time

APPROVED = {"corp-wifi": {"aa:bb:cc:00:11:22", "aa:bb:cc:00:11:23"}}
ssid_to_bssids = defaultdict(set)
last_seq = {}
deauth_times = defaultdict(list)

def alert(msg):
    print(f"🚨 {time.strftime('%H:%M:%S')}  {msg}")

def handle(pkt):
    if not pkt.haslayer(Dot11):
        return
    src = pkt.addr2

    # 1. Deauth flood
    if pkt.haslayer(Dot11Deauth):
        now = time.time()
        deauth_times[src] = [t for t in deauth_times[src] if now - t < 10]
        deauth_times[src].append(now)
        if len(deauth_times[src]) > 20:
            alert(f"Deauth flood from {src}")

    # 2. Rogue / evil-twin AP
    if pkt.haslayer(Dot11Beacon):
        ssid = pkt[Dot11Elt].info.decode(errors="ignore").lower()
        ssid_to_bssids[ssid].add(src)
        if ssid in APPROVED and src not in APPROVED[ssid]:
            alert(f"Rogue/evil-twin AP '{ssid}' on unapproved BSSID {src}")

    # 3. MAC spoofing via sequence-number regression
    seq = pkt.SC >> 4 if pkt.SC is not None else None
    if seq is not None and src in last_seq:
        gap = (seq - last_seq[src]) % 4096
        if gap > 3000:  # large backward-ish jump
            alert(f"Sequence anomaly for {src} (possible spoofing)")
    last_seq[src] = seq

sniff(iface="wlan0mon", prn=handle, store=0)
